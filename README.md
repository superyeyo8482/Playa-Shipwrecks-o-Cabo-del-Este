# Playa Shipwrecks – Cabo del Este · Real del Mar

**Landing page profesional para un terreno frente a la playa en East Cape, Los Cabos.**

Este repositorio contiene el código de una ficha técnica interactiva, diseñada para inversionistas extranjeros (en inglés), que explica:

- Esquema de coinversión 50/50
- Precio especial por volumen: **$195 USD/m²**
- Superficie total: **5,627.72 m²** (0.56 ha)
- Certidumbre legal mediante **Fideicomiso** (trust mexicano)
- Ventajas de ubicación en la zona de Shipwrecks / Cabo del Este
- Gráficos de plusvalía (tendencia de precios) y radar comparativo
- Contacto directo con asesor local y con un inversionista que ya compró ahí

## 🚀 Características técnicas

- **Video incrustado desde YouTube** – sin problemas de 404, responsivo.
- **Galería de 7 imágenes** (sin duplicados) – carga rápida y diseño grid.
- **Gráficos interactivos** con Chart.js (tendencia y radar).
- **Botones de WhatsApp** optimizados para móvil (tamaño reducido, alineados).
- **Botón de descarga de PDF** – análisis financiero completo.
- **Personaje animado “Inver‑tito”** que se pasea por la pantalla y cambia de mensaje.
- **Open Graph meta tags** para previsualización en redes sociales y WhatsApp.
- **Diseño 100% responsivo** (Tailwind CSS + media queries).
