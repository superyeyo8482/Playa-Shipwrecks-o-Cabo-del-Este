# Playa Shipwrecks – Cabo del Este · Real del Mar

**Landing page profesional para un terreno frente a la playa en East Cape, Los Cabos.**

Este repositorio contiene el código de una ficha técnica interactiva, diseñada para inversionistas extranjeros (en inglés), que explica:

- Esquema de coinversión 50/50
- Precio especial por volumen: **$195 USD/m²**
- Superficie total: **5,627.72 m²** (0.56 ha)
- Certidumbre legal mediante **Fideicomiso** (banco fiduciario)
- Ventajas de ubicación en la zona de **Shipwrecks / Cabo del Este**
- Gráficos de plusvalía (tendencia de precios) y radar comparativo
- Contacto directo con asesor local y con un inversionista que ya compró ahí

---

## 🛠️ Tecnologías utilizadas

| Tecnología | Propósito |
|------------|-----------|
| **React 18** | Componentes y estado (hooks: useState, useEffect, useRef) |
| **Babel (standalone)** | Compilar JSX en el navegador sin build step |
| **Tailwind CSS** | Estilos responsivos y diseño moderno (utility‑first) |
| **Chart.js** | Gráficos interactivos (línea de tendencia de precios y radar comparativo) |
| **chartjs-plugin-datalabels** | Mostrar etiquetas en gráficos (punto destacado) |
| **HTML5 / CSS3** | Estructura y animaciones (personaje “Inver‑tito”, keyframes) |
| **SVG + CSS** | Personaje original animado (casita + ruedas) |
| **Git + GitHub** | Control de versiones y alojamiento de assets |
| **Vercel** | Despliegue continuo y hosting gratuito |

## 📱 Características destacadas

- **Diseño 100% responsivo** (se adapta a móvil, tablet y escritorio)
- **Botones de WhatsApp con efecto neón y animación pulse** (dos contactos)
- **Personaje animado “Inver‑tito”** (mascota de inversiones) que se pasea por la pantalla y cambia de mensaje
- **Gráfico de radar** que compara la propiedad con el promedio de otras zonas de Los Cabos
- **Gráfico de tendencia de precios** con fuente basada en datos AMPI / SHF
- **Cálculo de coinversión 50/50** automático (total y por socio)
- **Explicación legal del fideicomiso** en inglés sencillo (ideal para extranjeros)
- **Enlaces directos a Google Maps** (ubicación exacta con distancias estimadas)
